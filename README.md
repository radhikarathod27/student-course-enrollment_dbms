Student–Course–Enrollment System – Fake Data Generator

This project generates a Student–Course–Enrollment database with 1000 fake Indian student records using Python and MySQL.

Features
Generates large-scale synthetic data  
Implements relational database structure  
Uses Python–MySQL connectivity  
Enforces primary and foreign key constraints  

Tables Created
Students  
Courses  
Enrollments  

Requirements
pip install mysql-connector-python

Run
python student_dbms.py

Output
MySQL database: schema_demo  
Tables: Student, Course, Enrollment  
View: Student_Enrollment_View  

Note
All data is fake and created for educational and academic use only.
